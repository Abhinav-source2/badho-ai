# Skills Gap Analysis Framework for AI Engineer Transition

## How to Use This Framework
Rate yourself 1-5 on each skill. Anything below 3 is a gap to close before
applying for senior AI engineer roles. Anything below 2 is a gap to close before
any AI engineer role.

## Core Technical Skills

### Python (weight: critical)
1 - Can write basic scripts
2 - Comfortable with OOP, can read others' code
3 - Strong: async/await, decorators, type hints, packaging
4 - Advanced: metaclasses, performance optimization, C extensions
5 - Expert: CPython internals, contributes to OSS Python projects

### LLM API Integration (weight: critical)
1 - Has read about LLMs but not built with them
2 - Can call OpenAI API, build basic chat
3 - Understands: tokens, context window, tool use, streaming, cost
4 - Advanced: multi-turn state, agentic loops, structured output
5 - Expert: fine-tuning, RLHF concepts, model evaluation methodology

### RAG Architecture (weight: high)
1 - Knows the acronym
2 - Understands the concept: retrieve + augment + generate
3 - Has built a RAG system: chunking, embedding, retrieval, prompt injection
4 - Advanced: reranking, hybrid search, query expansion, eval-driven tuning
5 - Expert: custom retrieval architectures, multi-modal RAG, production scale

### Vector Databases (weight: high)
1 - Never used one
2 - Understands concept of embeddings and cosine similarity
3 - Has used ChromaDB or FAISS in a project
4 - Advanced: Qdrant/Pinecone, metadata filtering, approximate nearest neighbor
5 - Expert: index tuning, HNSW parameters, production scaling

### Evaluation & Testing (weight: high)
1 - No AI eval experience
2 - Manual testing of AI outputs
3 - Automated evals: programmatic checks, LLM-as-judge
4 - Advanced: RAGAS, custom metrics, regression suites, A/B testing
5 - Expert: designed eval frameworks from scratch at scale

### Prompt Engineering (weight: medium)
1 - Casual user of ChatGPT
2 - Knows system prompts, basic few-shot
3 - Structured prompts: CoT, XML tags, JSON mode, tool schemas
4 - Advanced: prompt optimization, adversarial testing, injection prevention
5 - Expert: published research or widely-used prompt techniques

### MLOps (weight: medium)
1 - No MLOps experience
2 - Basic: knows what model versioning means
3 - Intermediate: MLflow or W&B, basic CI for ML
4 - Advanced: feature stores, model serving (Triton/vLLM), drift detection
5 - Expert: designed ML platform for large team

## Scoring Interpretation
- Average 4+: Ready for senior AI engineer roles at well-funded startups
- Average 3-4: Ready for mid-level AI engineer roles, stretch for senior
- Average 2-3: Ready for junior AI engineer roles, needs 3-6 months of focused
  projects to reach mid-level
- Average below 2: Backend-to-AI transition in progress — 6-12 months to
  job-ready

## Priority Learning Order
1. LLM API Integration — highest ROI, fastest to learn
2. RAG Architecture — most in-demand skill in 2024-2025
3. Evaluation — biggest differentiator in interviews
4. Vector Databases — learnable in a week with hands-on projects
5. Prompt Engineering — learnable through deliberate practice
6. MLOps — important for senior roles, less critical for entry/mid
