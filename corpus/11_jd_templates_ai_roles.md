# Job Description Templates — AI Engineer Roles 2024

## JD 1: AI Engineer — Series A Consumer App (Remote India)
Company: Early-stage AI startup, Series A, 20-person team
Salary: ₹30-50 LPA + equity

Responsibilities:
- Build and maintain LLM-powered features end-to-end
- Design and implement RAG pipelines for knowledge retrieval
- Own evaluation framework — define metrics, run experiments, ship improvements
- Optimize LLM costs — token efficiency, caching, model selection
- Collaborate with product to define what "working" means for AI features

Requirements:
- 3+ years Python, strong backend fundamentals
- 1+ year building with LLM APIs (OpenAI, Anthropic, or equivalent)
- Experience with vector databases (any: Pinecone, Qdrant, ChromaDB, pgvector)
- Understanding of RAG architecture — can build from scratch without frameworks
- Strong written communication — we are async-first

Nice to have:
- Fine-tuning experience (LoRA, QLoRA)
- Experience with LLM evaluation frameworks (RAGAS, custom)
- Open source contributions to AI tooling

## JD 2: ML Engineer — Series B SaaS (Bangalore/Hybrid)
Salary: ₹40-70 LPA

Responsibilities:
- Design ML pipelines for production — training, evaluation, deployment
- Own model serving infrastructure — latency, throughput, cost
- Build evaluation and monitoring systems for deployed models
- Partner with data team on feature engineering

Requirements:
- 4+ years ML engineering
- Strong Python, experience with PyTorch or TensorFlow
- MLOps experience: MLflow/Kubeflow/Vertex AI
- Production experience: have deployed models serving real traffic
